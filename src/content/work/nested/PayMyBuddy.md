## Objectifs

Créer une application de paiement entre particuliers avec authentification, gestion des contacts et enregistrement sécurisé des transactions.

## Réalisations

- inscription et authentification des utilisateurs ;
- ajout et suppression de relations entre utilisateurs ;
- création et consultation des transactions ;
- API REST complétée par une interface Thymeleaf ;
- modèle relationnel MySQL avec intégrité référentielle ;
- tests des contrôleurs et services avec JUnit et Mockito.

## Technologies utilisées

- Java 21 et Spring Boot
- Spring Security et Spring Data JPA
- Thymeleaf
- MySQL et H2 pour les tests
- JUnit, Mockito et JaCoCo
