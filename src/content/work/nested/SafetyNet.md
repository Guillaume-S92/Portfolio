## Objectifs

Développer le back-end de SafetyNet Alerts à partir de données JSON afin de fournir rapidement aux services de secours les informations utiles lors d'une intervention.

## Fonctionnalités

- habitants couverts par une caserne et répartition adultes/enfants ;
- alertes enfants, téléphone et inondation ;
- consultation des informations personnelles et médicales ;
- opérations de création, modification et suppression sur les personnes, casernes et dossiers médicaux ;
- journalisation applicative et tests automatisés.

## Technologies utilisées

- Java 21 et Spring Boot
- Spring Web et Jackson
- JUnit 5, Mockito et JaCoCo
- architecture MVC et API REST
